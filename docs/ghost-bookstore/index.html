幽靈書店

作者：葉霜降

大綱

大綱

在臺北一條即將被都更拆除的老巷弄裡，有一間沒有招牌的二手書店。店主是一位沉默寡言的老人，他聲稱每一本書裡都住著前任讀者的靈魂碎片。

大學畢業後找不到方向的林若晴，某天深夜誤入這間書店，翻開一本泛黃的日記體小說後，開始在夢中經歷書中角色的人生。她逐漸分不清自己的記憶與書中人物的記憶，而書店老人似乎對這一切早有預料。

隨著若晴讀完一本又一本的書，她發現這些書的前任讀者全都在現實中消失了。她必須在自己也被書吞噬之前，找到脫離的方法——或者接受成為下一本書的一部分。
